**BATTLEXARENA**

  BATTLEXARENA ek modern, mobile-first web application hai jo Nepali gamers ke liye tournaments host karne ke liye banaya gaya hai. Isme **Dark Neon Esports** theme ka use kiya gaya hai.

## ✨ Key Features
* 📱 **App-like UI:** Mobile users ke liye smooth experience.
* 🔥 **Multiple Games:** Free Fire, PUBG (BGMI), aur Ludo tournaments ka support.
* 💳 **Local Payments:** eSewa aur Khalti ke liye manual deposit aur withdrawal system.
* 🔐 **Secure Auth:** Firebase Authentication ke saath fast login/signup.
* 🛠️ **Admin Panel:** Matches manage karne aur payments approve karne ke liye powerful dashboard.

## 🚀 Tech Stack
* **Frontend:** HTML5, Tailwind CSS, JavaScript (Vanilla)
* **Backend:** Firebase Realtime Database
* **Hosting:** GitHub Pages / Firebase Hosting
